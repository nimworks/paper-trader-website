Frequently Asked Questions

What is Kaagzi?

Kaagzi is a paper trading app which allows you to take mock positions without actually entering a real trade.
Learn more about Kaagzi

How do I use Kaagzi?

Kaagzi is offered as a web-app & web-extension, either of which can be used independently. For the web-app, you would need to sign-up for API keys from at least one of the API providers supported by Kaagzi.
Kindly go through the provided guide to get started

Does Kaagzi use real-time live market data for mock trades?

YES. Kaagzi uses real-time live market data for your mock trades. This allows you to practice trading in real world scenario instead of a simulated one

Do I need an active account with any of the existing brokers?

NO. You do not need an active account with any of the brokers. But yes, in order to use the Kaagzi web application at kaagzi.in, you would need to register for the API made available free of cost by few of the well-known brokers

Does Kaagzi work offline?

NO. Since Kaagzi requires access to real-time live market data, it needs to be connected to the internet for your mock trade positions to be executed

Is Kaagzi a broker?

NO. Kaagzi is just a paper trading tool which facilitates you to take mock positions without staking real money.

Is Kaagzi available for Android and iOS?

YES. Kaagzi runs inside web-broser. Thus Kaagzi can be used on any deivce or OS where you have access to a web-browser

Does Kaagzi support Futures & Options?

YES. Kaagzi allows you to take mock positions for any of the tradable instruments available in India. It actually depends on the API provider you have signed up for

Is there a referral rewards program available?

NO. As of now we do not have any referral reward program
